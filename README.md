

# Show Me The Film
**영화 소개 어플**

## Description
영화를 장르별 또는 일일박스오피스, 주간박스오피스 별로 찾아볼 수 있음



## Development motive

시중에 나온 많은 영화중에 사용자가 보고싶은 영화만을 **빠르고** **쉽게** 찾아서 보기 위하여 개발


## Communication technique
- SQLite 데이터베이스를 사용하여 사용자의 정보 저장
	> 쿼리문을 사용하여 로그인 및 사용자의 정보를 불러옴
- Retrofit2 라이브러리를 사용하여 영화정보 파싱
	> 실시간 영화정보를 서버로부터 받아옴

## Used Language
- JAVA in Android Studio
